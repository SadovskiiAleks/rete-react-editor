База знаний Департамент BIM 2025

Версия: Корпоративные ресурсы + Renga + Pilot
Дата подготовки: 22.05.2026

Как открыть:
1. Распакуйте архив.
2. Откройте файл index.html двойным кликом.
3. Сайт работает локально на ПК без сервера.

Состав данных:
- Корпоративные ресурсы + Департамент BIM: 30 материалов.
- Renga: 57 материалов.
- Pilot: 66 материалов.
Итого: 153 материала.

Файл для редактирования содержимого:
- data.js

Файлы, которые обычно не нужно редактировать:
- index.html — структура страницы.
- style.css — оформление.
- script.js — логика поиска, фильтров и карточек.
- data.json — копия данных для импорта/автоматизации.

Примечание по Pilot:

- Полезные ссылки
- Руководства
